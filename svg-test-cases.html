Instrukce pro testování:

  1. Ujistěte se, že je extension načten (reload extension v chrome://extensions/)
  2. Najeďte myší na každou ikonu - měl by se objevit popup s buttony
  3. Stiskněte CMD+click (nebo klikněte na download button) pro stažení SVG
  4. Otevřete stažený SVG v editoru nebo browseru
  5. Ověřte:
    • ✅ Žádné class atributy
    • ✅ Všechny elementy mají fill nebo stroke
    • ✅ Správný xmlns a viewBox
    • ✅ Validní XML bez errorů
    • ✅ Ikona se zobrazí správně
